# Weather App

A simple web application to detect and display weather information for cities around the world.

## Features

- Search for weather by city name
- Display current temperature, humidity, wind speed, and weather description
- Visual weather icons for different conditions (sunny, cloudy, rainy, etc.)
- Responsive design

## Usage

1. Open `index.html` in a web browser.
2. Enter a city name in the search box.
3. Click the search button or press Enter.
4. View the current weather information for the entered city.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeatherMap API (assumed, based on typical weather apps)

## Setup

No additional setup required. Just open the HTML file in a browser. Ensure you have an internet connection for API calls.
